# Examples

This folder is for concrete MicroPatch examples and reference material.

Use examples to study:

- how a good `micropatch.md` is structured
- how anchors describe file purpose and likely integration points
- how Definition of Done is written so feature preservation can be verified
- how upstream overlap clues are documented for future applies

Current example:

- `micropatch-auth.md`

Keep this folder small and curated. One strong example is usually better than many weak ones.
